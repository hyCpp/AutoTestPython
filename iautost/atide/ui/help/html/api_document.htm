AutoTest Api简介

AutoTest Api主要包括以下几个方面：车辆信号操作、设备画面操作和判断及环境部署相关等。

一、车辆信号操作

acc_on()

acc_off()

bup_on()

bup_off()

rev_on()

rev_off()

boot()

reboot()

二、画面操作和判断

keyevent(key)，其中key的取值为：KEY_HOME、KEY_MENU、KEY_VOLUMNP_UP、KEY_VOLUMN_DOWN等

touch(pic)

touch_if(pic)

touch_in(pic, pic2)

touch_or(pic, pic2)

long_touch(pic, duration=2.0)

swipe(pic, pic2)

flick(pic, dir)，其中dir的取值为：DIR_UP、DIR_DOWN、DIR_LEFT、DIR_RIGHT、DIR_UP_LEFT、DIR_UP_RIGHT、DIR_DOWN_LEFT、DIR_DOWN_RIGHT

pinch(pos, scale)    #for example, pinch((400,240), 0.9)

exists(pic)    #return true or false

assert_exists(pic)

assert_not_exists(pic)

poco(**kwargs).click()      #for example, poco(text='Source').click()

poco(**kwargs).exists()     #return true or false

poco(**kwargs).wait_for_appearance()

poco(**kwargs).wait_for_disappearance()

三、环境部署相关

auto_setup()      #安装AutoTest相关库

connect_device(device_uri)     #连接设备

disconnect_device(device_uri)     断开设备
