一、硬件准备

1.PC（Windows）/实机/TBox（二代盒子）

2.接线：串口/网线/TBox连接线

二、PC端操作手顺

1.安装python

（1）安装atlib\dependency\python-3.5.4.exe

（2）把Python安装目录加入到环境变量PATH中

 以上操作完成后，在命令行窗口下输入“python”提示python 3.5.4..，表示python安装完成且配置成功。

 

2.安装pyqt&qt

atlib\dependency\\PyQt5-5.6-gpl-Py3.5-Qt5.6.0-x32-2.exe

 

3.安装python第三方库

    命令行窗口下，通过cd指令跳到atlib目录，在该目录下有install.py文件，输入python install.py命令，回车。系统将一键安装第三方库。安装完毕后，命令行窗口提示“Finished processing dependencies for autost==0.10”，python的第三方库安装成功。

 

4.部署iauto2test服务至实机

（1）python machine\t0_power_on.py（如实机已经启动，该步骤可跳过）

（2）python machine\t0_setup.py

（3）python machine\t0_reboot.py

三、启动IDE

方式1：命令行窗口下,通过cd命令跳到atide目录下, 执行python main.py

方式2：第一次运行，可以采用以下方式运行：

        对atide目录下的main.py右键“打开方式”，“选择默认程序”， “浏览”，选择python安装目录下的python.exe, 确定。以后再运行，直接双击main.py即可。
